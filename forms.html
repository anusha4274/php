Working with Forms in PHP

Learn how to handle HTML forms and user input in PHP.

Working with HTML Form Controls and PHP

HTML forms allow users to submit data to the server. PHP can process this data. Here’s an example of a simple form:

                <form action="process.php" method="post">
                    <label for="name">Name:</label>
                    <input type="text" id="name" name="name">
                    <input type="submit" value="Submit">
                </form>

Superglobal Variables

PHP provides superglobal variables that are accessible from any scope. The most commonly used superglobals for form handling are:

  • $_GET: Used to collect data sent in the URL query string.
  • $_POST: Used to collect data submitted via HTTP POST method.
  • $_REQUEST: Combines $_GET, $_POST, and $_COOKIE.

Superglobal Array

Superglobal arrays are built-in PHP arrays that hold information about the request, session, and environment. Here’s how to access data from a form submission:

                <?php
                // process.php
                if ($_SERVER["REQUEST_METHOD"] == "POST") {
                    $name = $_POST["name"];
                    echo "Hello, " . htmlspecialchars($name) . "!";
                }
                ?>

Importing User Input

When a form is submitted, the data can be imported into PHP using the superglobal arrays. Here’s how it works:

                <form action="process.php" method="post">
                    <label for="email">Email:</label>
                    <input type="email" id="email" name="email">
                    <input type="submit" value="Submit">
                </form>
                <?php
                // process.php
                if ($_SERVER["REQUEST_METHOD"] == "POST") {
                    $email = $_POST["email"];
                    echo "Your email is: " . htmlspecialchars($email);
                }
                ?>

Accessing User Input

Access user input through the superglobal arrays. Always validate and sanitize the input to prevent security vulnerabilities:

                <?php
                if ($_SERVER["REQUEST_METHOD"] == "POST") {
                    $name = $_POST["name"] ?? ''; // Using null coalescing operator
                    $name = trim($name); // Remove whitespace
                    if (!empty($name)) {
                        echo "Welcome, " . htmlspecialchars($name) . "!";
                    } else {
                        echo "Name cannot be empty!";
                    }
                }
                ?>
