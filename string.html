Strings in PHP

Master the art of string manipulation in PHP.

String Interpolation with Curly Braces

String interpolation allows you to embed variables within double-quoted strings. To avoid ambiguity, you can use curly braces:

                <?php
                $name = "John";
                echo "Hello, {$name}!"; // Outputs: Hello, John!
                ?>

Characters and String Indexes

Strings in PHP are indexed, allowing you to access individual characters:

                <?php
                $text = "Hello";
                echo $text[0]; // Outputs: H
                echo $text[1]; // Outputs: e
                ?>

String Operators

PHP provides several operators for manipulating strings:

  • Concatenation Operator (.): Used to combine two strings.
    •                     <?php
                    $firstName = "John";
                    $lastName = "Doe";
                    $fullName = $firstName . " " . $lastName; // Outputs: John Doe
                    echo $fullName;
                    ?>
  • Concatenation Assignment Operator (.=): Appends a string to another string.
    •                     <?php
                    $greeting = "Hello";
                    $greeting .= " World!"; // Outputs: Hello World!
                    echo $greeting;
                    ?>

Heredoc

The `heredoc` syntax allows you to create multi-line strings without needing to escape quotes:

                <?php
                $text = <<<EOT
                This is a heredoc string.
                It can span multiple lines.
                EOT;
                echo $text;
                ?>

String Functions

PHP offers a variety of built-in string functions. Here are a few common ones:

  • strlen(): Returns the length of a string.
    •                     <?php
                    $length = strlen("Hello"); // Outputs: 5
                    ?>
  • strtoupper(): Converts a string to uppercase.
    •                     <?php
                    echo strtoupper("hello"); // Outputs: HELLO
                    ?>
  • strtolower(): Converts a string to lowercase.
    •                     <?php
                    echo strtolower("HELLO"); // Outputs: hello
                    ?>

Formatting Strings

To format strings, you can use the `printf()` or `sprintf()` functions:

                <?php
                $name = "John";
                $age = 25;
                printf("%s is %d years old.", $name, $age); // Outputs: John is 25 years old.
                ?>

Comparing and Searching Strings

PHP provides functions for comparing and searching within strings:

  • strcmp(): Compares two strings (case-sensitive).
    •                     <?php
                    $result = strcmp("abc", "ABC"); // Outputs a positive number
                    ?>
  • strstr(): Finds the first occurrence of a substring.
    •                     <?php
                    $text = "Hello World";
                    echo strstr($text, "World"); // Outputs: World
                    ?>
  • strpos(): Finds the position of the first occurrence of a substring.
    •                     <?php
                    $position = strpos("Hello World", "World"); // Outputs: 6
                    ?>

Substrings

To extract a substring, you can use the `substr()` function:

                <?php
                $text = "Hello World";
                echo substr($text, 0, 5); // Outputs: Hello
                ?>
