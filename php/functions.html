Functions in PHP

Explore the power of functions to organize your code efficiently.

Function Definition

A function is a block of code that performs a specific task. Functions help in code reusability and organization. The basic syntax of a function is:

                <?php
                function functionName() {
                    // Code to be executed
                }
                ?>

Creating and Invoking User-Defined Functions

To create a user-defined function, define it using the syntax above and invoke it by calling its name:

                <?php
                function greet() {
                    echo "Hello, World!";
                }
                
                greet(); // Calling the function
                ?>

Formal Parameters vs. Actual Parameters

Parameters are variables passed to functions. There are two types:

  • Formal Parameters: Defined in the function declaration.
    •                     <?php
                    function add($a, $b) { // $a and $b are formal parameters
                        return $a + $b;
                    }
                    ?>
  • Actual Parameters: Values passed to the function when called.
    •                     <?php
                    $sum = add(5, 10); // 5 and 10 are actual parameters
                    echo $sum; // Outputs 15
                    ?>

Function and Variable Scope

The scope of a variable determines where it can be accessed. Variables defined inside a function have local scope, while those defined outside have global scope:

                <?php
                $globalVar = "I am global"; // Global variable

                function test() {
                    $localVar = "I am local"; // Local variable
                    echo $localVar;
                }

                test(); // Outputs "I am local"
                echo $globalVar; // Outputs "I am global"
                ?>

Recursion

Recursion is a programming technique where a function calls itself. It's useful for solving problems that can be broken down into smaller, similar problems. Hereâ€™s an example of a recursive function to calculate the factorial of a number:

                <?php
                function factorial($n) {
                    if ($n <= 1) {
                        return 1; // Base case
                    } else {
                        return $n * factorial($n - 1); // Recursive call
                    }
                }

                echo factorial(5); // Outputs 120
                ?>

Library Functions

PHP provides a rich set of built-in functions, known as library functions, to perform various tasks. Some common library functions include:

  • strlen(): Returns the length of a string.
  • strpos(): Finds the position of the first occurrence of a substring in a string.
  • array_merge(): Merges one or more arrays.
  • date(): Formats a date and time.

Example of using a library function:

                <?php
                $text = "Hello";
                echo strlen($text); // Outputs 5
                ?>
