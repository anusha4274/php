Working with the File System in PHP

Learn how to manage files and directories using PHP.

Understanding PHP File Permissions

File permissions dictate who can read, write, or execute a file. In PHP, you can use the chmod() function to change file permissions:

                <?php
                chmod("example.txt", 0755); // Set permissions to read/write/execute for owner, read/execute for others
                ?>

Opening and Closing a File

Use the fopen() function to open a file. You can specify the mode of opening (e.g., read, write):

                <?php
                $file = fopen("example.txt", "r"); // Open file for reading
                // Always close the file after use
                fclose($file);
                ?>

File Reading and Writing Functions

Reading from a File

To read from a file, you can use functions like fread() or fgets():

                <?php
                $file = fopen("example.txt", "r");
                $content = fread($file, filesize("example.txt")); // Read entire file
                fclose($file);
                echo $content;
                ?>

Writing to a File

To write to a file, open it in write mode using w or append mode using a:

                <?php
                $file = fopen("example.txt", "a"); // Open file for appending
                fwrite($file, "New line of text.\n"); // Write to file
                fclose($file);
                ?>

File System and Directory Functions

PHP provides several functions to interact with the file system:

Creating a Directory

You can create a directory using the mkdir() function:

                <?php
                mkdir("new_directory"); // Create a new directory
                ?>

Deleting a Directory

To delete a directory, use the rmdir() function:

                <?php
                rmdir("new_directory"); // Remove the directory
                ?>

Checking If a File Exists

You can check if a file exists using file_exists():

                <?php
                if (file_exists("example.txt")) {
                    echo "File exists.";
                } else {
                    echo "File does not exist.";
                }
                ?>

Getting File Information

Use filesize() to get the size of a file:

                <?php
                $size = filesize("example.txt");
                echo "File size: " . $size . " bytes.";
                ?>

Listing Files in a Directory

To list files in a directory, use the scandir() function:

                <?php
                $files = scandir("some_directory");
                foreach ($files as $file) {
                    echo $file . "<br>";
                }
                ?>
