Arrays in PHP

Learn how to work with arrays, a fundamental data structure in PHP.

PHP Arrays

Arrays are used to store multiple values in a single variable. PHP supports three types of arrays:

  • Indexed Arrays: Arrays with numeric index.
  • Associative Arrays: Arrays with named keys.
  • Multidimensional Arrays: Arrays containing one or more arrays.

Creating Arrays

You can create an array using the array() function or the shorthand [] syntax:

                <?php
                // Indexed array
                $fruits = array("Apple", "Banana", "Cherry");
                $colors = ["Red", "Green", "Blue"]; // Short syntax
                ?>

Accessing Array Elements

Access array elements using their index:

                <?php
                echo $fruits[0]; // Outputs: Apple
                echo $colors[1]; // Outputs: Green
                ?>

Multidimensional Arrays

Multidimensional arrays are arrays containing other arrays:

                <?php
                $students = [
                    "John" => ["age" => 25, "major" => "CS"],
                    "Jane" => ["age" => 22, "major" => "Math"],
                ];
                echo $students["John"]["major"]; // Outputs: CS
                ?>

Inspecting Arrays

Use functions like print_r() or var_dump() to inspect array contents:

                <?php
                print_r($fruits);
                var_dump($students);
                ?>

Deleting from Arrays

To delete an element from an array, use the unset() function:

                <?php
                unset($fruits[1]); // Removes Banana
                print_r($fruits); // Outputs: Array ( [0] => Apple [2] => Cherry )
                ?>

Iterating with each() and foreach()

The foreach() loop is a simple way to iterate over arrays:

                <?php
                foreach ($fruits as $fruit) {
                    echo $fruit . "<br>"; // Outputs each fruit
                }
                ?>

Note: The each() function is deprecated and not recommended for use in new code.

Iterative Functions

PHP provides several functions for manipulating array pointers:

  • current(): Returns the current element in an array.
    •                     <?php
                    echo current($fruits); // Outputs: Apple
                    ?>
  • next(): Advances the array pointer to the next element.
    •                     <?php
                    next($fruits); // Moves to Banana
                    echo current($fruits); // Outputs: Cherry
                    ?>
  • prev(): Moves the array pointer to the previous element.
    •                     <?php
                    prev($fruits); // Moves back to Banana
                    echo current($fruits); // Outputs: Banana
                    ?>
  • reset(): Moves the array pointer to the first element.
    •                     <?php
                    reset($fruits); // Moves back to Apple
                    echo current($fruits); // Outputs: Apple
                    ?>
  • end(): Advances the array pointer to the last element.
    •                     <?php
                    end($fruits); // Moves to the last element
                    echo current($fruits); // Outputs: Cherry
                    ?>
