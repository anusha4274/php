PHP Basics

Get started with the essentials of PHP programming.

Tokens

Tokens are the smallest units of PHP code that have meaning. PHP recognizes different types of tokens, including:

  • Variables: Represented by a `$` sign followed by the variable name.
  • Operators: Symbols that perform operations on variables and values.
  • Keywords: Reserved words that have special meaning in PHP (e.g., `if`, `else`, `while`).
  • Strings: Sequences of characters.

Variables

Variables in PHP are used to store data, which can be referenced and manipulated throughout the script. Variables are declared using the `$` sign:

                <?php
                $name = "John";
                $age = 25;
                ?>

Variable Scope

Variable scope determines where a variable can be accessed within a script:

  • Global Scope: Variables declared outside any function. Accessible from anywhere in the script.
  • Local Scope: Variables declared inside a function. Accessible only within that function.
  • Static Scope: Local variables that maintain their value even after the function exits.

Constants

Constants are like variables but cannot be changed after they are defined. They are defined using the `define()` function:

                <?php
                define("SITE_NAME", "PHP Learning Hub");
                echo SITE_NAME;
                ?>

Data Types

PHP supports several data types, which can be classified into four categories:

  • Scalar Types:
    • String: A sequence of characters.
    • Integer: Whole numbers.
    • Float: Decimal numbers.
    • Boolean: True or false values.
  • Compound Types:
    • Array: A collection of values.
    • Object: Instances of classes.

Number Handling in PHP

PHP provides various functions for handling numbers, such as:

  • abs(): Returns the absolute value.
  • round(): Rounds a number to the nearest integer.
  • rand(): Generates a random integer.
  • floor(): Rounds down to the nearest integer.
  • ceil(): Rounds up to the nearest integer.

Operands, Operators, and Expressions

Operands are the values on which operators act. Operators perform operations on operands. An expression is a combination of variables, constants, and operators that yields a value:

                <?php
                $sum = $a + $b; // $a and $b are operands, + is the operator
                ?>

Operator Precedence

Operator precedence determines the order in which operations are performed in an expression. For example:

                <?php
                $result = 10 + 20 * 5; // Result is 110, not 150
                ?>

In this case, multiplication is performed before addition.

Comments

Comments are notes in the code for clarity, which are ignored by the PHP engine. There are two types of comments in PHP:

  • Single-line Comments: Start with `//` or `#`.
  • Multi-line Comments: Enclosed between `/*` and `*/`.

echo and print Statements

The `echo` and `print` statements are used to output data to the screen:

                <?php
                echo "Hello, World!";
                print "This is PHP!";
                ?>

Both can be used interchangeably, but `echo` can take multiple parameters while `print` can only take one and returns 1.
